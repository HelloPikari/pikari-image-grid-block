A customizable image grid block for WordPress that displays four images in various layout patterns.

== Description ==

Create beautiful, responsive image grids with this comprehensive WordPress Gutenberg block plugin. Perfect for portfolios, galleries, and visual content organization.

**Key Features:**

* **Multiple Layout Options** - Choose from asymmetric, equal grid, or masonry layouts
* **Responsive Images** - Uses WordPress's built-in image functions for proper srcset and lazy loading
* **Customizable Styling** - Adjust gap between images and border radius
* **Caption Support** - Optional captions for accessibility
* **Resizable** - Visual resize handles in the editor
* **Dynamic Rendering** - Server-side rendering for optimal performance
* **Block Variations** - Three pre-configured layout styles available as block variations
* **Flexible Sizing** - Support for custom width and height with multiple unit options

**Perfect for:**
* Image galleries
* Portfolio showcases
* Product displays
* Visual content organization
* Any grid-based image layouts

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pikari-image-grid-block/`, or install through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. In the block editor, add the "Image Grid" block from the Pikari category
4. Select up to four images using the sidebar controls
5. Choose a layout style and customize styling options

== Frequently Asked Questions ==

= How many images can I add to the grid? =

The Image Grid block is designed to display exactly four images in various layout patterns (asymmetric, equal grid, or masonry).

= Does it work with my theme? =

Yes! The plugin uses WordPress core layout features and follows coding standards for broad theme compatibility. It respects your theme's styling and integrates seamlessly.

= Are the images responsive? =

Absolutely! The plugin uses WordPress's built-in image functions for proper srcset generation and lazy loading, ensuring optimal performance across all devices.

= Can I customize the layout and styling? =

Yes! You can adjust the gap between images, border radius, enable/disable captions, and choose from three different layout styles. The block also supports custom width and height settings.

= Does it support captions and alt text? =

Yes! The block includes full caption support and allows you to set alt text for each image to ensure accessibility compliance.
